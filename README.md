<h1> Tissue Growth </h1>
<h3>A tissue growth model recreation using NetLogo.</h3>

<h2> WHAT IS IT? </h2>
Modelling and Analysing the Growth of a Network.

The biological notion is that of growth and structure formation by a cell (containing a “nucleus”) that reproduces by splitting off daughter cells, which are formed at defined positions (lower left and lower right of the parent cell).

<h2> HOW IT WORKS </h2>
Biological Conception.
The biological notion is that of growth and structure formation by a cell (containing a “nucleus”) that reproduces by splitting off daughter cells, which are formed at defined positions (lower left and lower right of the parent cell).

After this, they behave as parent cells and repeat the cycle of separation and positioning. However, daughter cells that end up at the same location experience competition for space and resources and as a result become infertile.

The outcome of the process is a tree-shaped “tissue” of green and red patches (respectively reproducing and non-reproducing cells).


<h2> HOW TO USE IT </h2>
Click the Set-up button to set up the initial agent.
Click ‘go-once’ to run the program incrementally(+1).
Click ‘go’ to run the program.
Move the ‘#generations’ slider to specify the number of generations outputted.
The ‘show-patch-color’ switch clears the patches by setting their color to black.
When the ‘show-circle?’ switch is ON, the nodes of the network is arranged as a circle.
Note: The “to go” stops after it has been checked if generationstep = #generations.
